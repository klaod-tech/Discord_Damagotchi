# 일정봇 — ML 계획

---

## 현재: 규칙 기반

- 반복 여부는 사용자가 직접 명시 (자연어에 "매일/매주" 포함 시 GPT 파싱)
- 알림은 고정 5분 간격 체크

---

## 향후: 반복 패턴 자동 감지

```
동일 제목/요일의 일정이 2회 이상 반복되면:
  → "매주 {요일} {제목}으로 등록할까?" 제안
  → 유저 승인 시 is_repeat=True 업데이트
```

---

## 향후: 일정 × 식사 연계

- 운동 일정 전날 → 식사봇에 단백질 식사 추천 알림
- 외식 일정 → 식사봇에 "오늘 외식이네! 칼로리 주의해줘" 알림

---

## 향후: 개인 패턴 분석

- 자주 등록하는 일정 유형 (병원/운동/약속) 분류
- 주간 패턴 → 먹구름봇 하루정리에 "이번 주 {N}개 일정 완료!" 표시
